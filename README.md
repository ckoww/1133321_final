# 飯店房務與客房紀錄管理系統 (Hotel Room Management System)

透過直觀的視覺化介面，讓使用者能快速掌握各客房的即時狀態，並完成房務交接與退房日期的追蹤，有效提升飯店客房管理的營運效率。

### ✨ 核心功能與特色
* **視覺化房況總覽 (Dashboard)**：系統啟動時會動態生成客房矩陣，並以直覺的顏色（綠色空房、紅色入住、黃色待清潔、灰色維修）標示房況，一目了然。
* **智慧防呆與 UI 連動機制**：當房間切換為「空房」或「維修中」時，系統會自動鎖定並重置「退房日期」與「客房服務指示」，防止人員誤填不合理的交接資訊。
* **客房服務狀態追蹤**：支援「請勿打擾 (DND)」與「請即打掃 (MUR)」標籤，並直接顯示於房況總覽按鈕上，無須點擊即可獲取房務指示。
* **穩定的資料持久化 (SQLite)**：採用 `System.Data.SQLite` 進行本機資料庫存取，具備完善的資料庫初始化與防錯機制。

## 💻 執行說明 (How to Run)
為符合專案繳交規範，本專案已移除 `bin`、`obj` 等編譯產生之執行檔與暫存檔。請依照以下步驟於本機執行：

1. **開啟專案**：使用 Visual Studio 開啟本專案方案檔 (`_1133321_final.sln`)。
2. **還原 NuGet 套件**：本專案依賴 `System.Data.SQLite.Core`。Visual Studio 預設會在編譯時自動還原遺失的 NuGet 套件；若無，請至「NuGet 套件管理員」手動點擊還原。
3. **編譯與執行**：點擊上方選單的 **建置 > 重建方案**，接著按下 **F5 啟動** 程式。
4. **資料庫自動初始化**：程式首次執行時，若偵測不到 `.db` 資料庫檔案，會**自動於輸出目錄 (`bin\Debug`) 生成全新的資料庫與資料表**，並預設匯入 6 間客房的初始測試資料，評分委員無須手動架設任何資料庫環境即可直接測試。
