# 飯店房務與客房紀錄管理系統 (Hotel Room Management System)

透過直觀的視覺化介面，讓使用者能快速掌握各客房的即時狀態，並完成房務交接與退房日期的追蹤，有效提升飯店客房管理的營運效率。

### ✨ 核心功能與特色
* **視覺化房況總覽 (Dashboard)**：系統啟動時會動態生成客房矩陣，並以直覺的顏色（綠色空房、紅色入住、黃色待清潔、灰色維修）標示房況，一目了然。
* **智慧防呆與 UI 連動機制**：當房間切換為「空房」或「維修中」時，系統會自動鎖定並重置「退房日期」與「客房服務指示」，防止人員誤填不合理的交接資訊。
* **客房服務狀態追蹤**：支援「請勿打擾 (DND)」與「請即打掃 (MUR)」標籤，並直接顯示於房況總覽按鈕上，無須點擊即可獲取房務指示。
* **穩定的資料持久化 (SQLite)**：採用 `System.Data.SQLite` 進行本機資料庫存取，具備完善的資料庫初始化與防錯機制。

---

## 🚀 開發環境與技術
* **開發語言**：C# (.NET Framework)
* **UI 框架**：Windows Forms (WinForms)
* **資料庫**：SQLite (`System.Data.SQLite.Core`)
